# Evolution automatisation IPC avec le logiciel R

En octobre 2025, le projet Automatisation IPC a été décliné pour toutes les régions du Maroc.
A ce jour le projet calcule à partir du fichier excel IPC transmis par le HCP central à Rabat
- l'IPC alimentaire et non alimentaire pour chaque ville en fonction de es pondérations
- alimente un fichier historique de l'IPC avec l'alimentaire et non alimentaire depuis janvier 2017
- produit des documents html d'aide à la rédaction des notes mensuelles

Le projet peut se poursuivre sur les pistes suivantes :
- produire le fichier excel qui sert à alimenter la BDS de chaque région
- produire les graphiques pour la note mensuelle
- produire le tableau diffusé dans la note mensuelle
- produire un modéle de document word avec graphique, tableau et mise en forme. L'idée est que la personne en charge de la rédaction de la note mensuelle n'est qu'à intégrer le texte d'analyse dans le dcument word
- envisager la production de la note annuelle
